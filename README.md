# 简易记账本

简洁易用的个人记账应用

## 功能特点

- ✅ 记录收入和支出
- ✅ 自定义分类
- ✅ 数据统计图表
- ✅ 数据导出/导入备份
- ✅ 完全离线可用
- ✅ 数据本地存储，保护隐私

## 在线使用

直接打开 `index.html` 即可使用

## 安装到手机

### 方法1：添加到主屏幕（PWA）
1. 在手机浏览器中打开应用
2. 点击浏览器菜单 → "添加到主屏幕"

### 方法2：打包成APK
1. 将此项目部署到GitHub Pages
2. 访问 https://www.pwabuilder.com/
3. 输入你的GitHub Pages地址
4. 点击生成APK并下载
